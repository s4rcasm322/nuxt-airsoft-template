# nuxt-airsoft-template

## Запуск и билд проекта

```bash
# Установка зависимостей
$ npm install

# Запуск dev версии с hot-reload
$ npm start

# Сборка проекта и его запуск
$ npm run build
$ npm run build-start

# Сгенерировать статичный проект
$ npm run generate
```
